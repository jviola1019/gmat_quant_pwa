# GMAT Study PWA v3

A modern, offline-first study companion. Created by Joe Viola with assistance from ChatGPT-5.2 & Gemini 3 Pro.

## Features
* **Beams Background**: A fluid 3D background powered by vanilla Three.js.
* **Multi-Step Quizzes**: Questions are broken down into logical steps. Get a step wrong twice, and the quiz resets to enforce mastery.
* **Flashcards**: 3D flip cards with question/answer format.
* **Cheatsheet**: Unified searchable rules.
* **Material Design 3**: Clean, dark-themed UI.

## How to Use
1.  Open `index.html`.
2.  Choose **Quiz** to practice multi-step problems.
3.  Choose **Flashcards** for quick recall.

## Data
All content is in `data/content.json`.
